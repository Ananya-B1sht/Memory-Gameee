# Memory Game 🎮

### 📌 Description
This is a simple memory card matching game built using HTML, CSS, and JavaScript.  
The player flips cards to find matching pairs while keeping track of moves and time.

---

### ✨ Features
- Three difficulty levels: Easy, Medium, Hard  
- Card flip and match functionality  
- Move counter  
- Timer  
- Hint option to reveal cards briefly  
- Restart button  
- Basic win animation  

---

### 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Docker  
- Nginx  

---

### ▶️ Run Locally
Open the `index.html` file in your browser.

---

### 🐳 Run Using Docker

docker build -t memory-game .  
docker run -p 8080:80 memory-game  

---

### 🌐 Open in Browser
http://localhost:8080
